# Spatial Assessments Airtable table

Create one table named `Spatial Assessments` in the same Airtable base used by the website.

Use these exact field names:

| Field | Recommended type |
| --- | --- |
| Assessment ID | Single line text (primary field) |
| Created At | Single line text |
| Company | Single line text |
| Contact | Single line text |
| Email | Email or single line text |
| Phone | Phone or single line text |
| Needs | Long text |
| Scene | Single line text |
| Stage | Single line text |
| Sites | Single line text |
| People / Assets | Single line text |
| Monthly Users | Single line text |
| Update Frequency | Single line text |
| Timeline | Single line text |
| Accuracy | Single line text |
| Notes | Long text |
| Status | Single line text |

Keep `Status` as single line text for the first version. The admin page uses:

`New`, `Contacted`, `Qualified`, `Quoted`, `Closed`

If the Airtable token already has read/write access to this base, no new token is required.
